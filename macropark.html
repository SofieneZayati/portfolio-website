Project Overview

🎯 Problem Statement

Traditional parking systems lack real-time monitoring and flexible access control. Manual processes lead to inefficiencies, long wait times, and poor user experience in busy parking facilities.

💡 Solution

An intelligent parking system combining License Plate Recognition (LPR) with BLE mobile app fallback, featuring ESP32-controlled barriers, MQTT communication, and robust failover mechanisms.

🚀 Impact

Streamlined parking operations, reduced wait times, enhanced security through dual authentication methods, and improved user experience with seamless mobile integration.

Key Features

🔍 License Plate Recognition

Primary authentication method using advanced LPR technology for seamless vehicle identification and automatic barrier control with 98.7% accuracy rate.

📱 BLE Mobile App Fallback

Secondary authentication via Bluetooth Low Energy mobile app when LPR fails or for manual override scenarios, ensuring 100% access reliability.

🚧 ESP32 Barrier Control

Robust hardware control system managing entry/exit barriers with real-time status monitoring, feedback, and emergency override capabilities.

📡 MQTT/BLE Communication

Dual communication protocols ensuring reliable data transmission between devices and cloud services with automatic failover.

🌐 Hybrid Network Failover

Ethernet/WiFi redundancy ensuring continuous operation even during network connectivity issues with automatic switching.

🔄 OTA Updates

Over-the-air firmware updates enabling remote system maintenance, security patches, and feature deployments without physical access.

Technical Implementation

// ESP32 Barrier Control Example
#include <WiFi.h>
#include <PubSubClient.h>
#include <BluetoothSerial.h>

#define BARRIER_PIN 23
#define STATUS_LED 2

class BarrierController {
private:
    int barrierPin;
    bool isOpen;
    unsigned long lastToggleTime;
    
public:
    BarrierController(int pin) : barrierPin(pin), isOpen(false), lastToggleTime(0) {
        pinMode(barrierPin, OUTPUT);
        pinMode(STATUS_LED, OUTPUT);
        digitalWrite(barrierPin, LOW);
    }
    
    void openBarrier() {
        if (millis() - lastToggleTime > 3000) { // Debounce
            digitalWrite(barrierPin, HIGH);
            digitalWrite(STATUS_LED, HIGH);
            isOpen = true;
            lastToggleTime = millis();
            Serial.println("Barrier opened");
        }
    }
    
    void closeBarrier() {
        if (millis() - lastToggleTime > 3000) { // Debounce
            digitalWrite(barrierPin, LOW);
            digitalWrite(STATUS_LED, LOW);
            isOpen = false;
            lastToggleTime = millis();
            Serial.println("Barrier closed");
        }
    }
    
    void toggleBarrier() {
        if (isOpen) closeBarrier();
        else openBarrier();
    }
};

// MQTT Callback Function Example
void callback(char* topic, byte* payload, unsigned int length) {
    Serial.print("Message arrived [");
    Serial.print(topic);
    Serial.print("] ");
    
    String message;
    for (int i = 0; i < length; i++) {
        message += (char)payload[i];
    }
    Serial.println(message);
    
    if (String(topic) == "parking/barrier/control") {
        if (message == "open") {
            barrierController.openBarrier();
        } else if (message == "close") {
            barrierController.closeBarrier();
        }
    }
}
